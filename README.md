# 🍅 Pomodoro Timer

> **Claude Code 연구 목적으로 제작된 베타 프로젝트입니다.**  
> 이 프로젝트는 **Claude AI (claude.ai)** 와의 대화를 통해 코드를 생성하고, GitHub Pages로 배포하는 전 과정을 테스트하기 위해 만들어졌습니다.

## 🔗 Live Demo

👉 [https://nitally0715.github.io/pomodoro-timer_Claude_Code_test/](https://nitally0715.github.io/pomodoro-timer_Claude_Code_test/)

## ✨ 기능

- **집중 / 짧은 휴식 / 긴 휴식** 모드 전환
- 원형 프로그레스 링으로 남은 시간 시각화
- 세션 완료 시 자동으로 다음 모드 전환 (4세션마다 긴 휴식)
- 완료 세션 수, 총 집중 시간, 연속 세션 통계 표시
- 일시정지 / 재개 / 초기화 / 건너뛰기 지원

---

## 🛠️ 기술 스택

- HTML / CSS / Vanilla JS (외부 라이브러리 없음)
- Google Fonts (DM Mono, Instrument Serif)
- GitHub Pages로 배포

## 🚀 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 바로 열면 됩니다.

```bash
git clone https://github.com/nitally0715/pomodoro-timer_Claude_Code_test.git
cd pomodoro-timer_Claude_Code_test
open index.html  # macOS
# 또는 index.html을 브라우저에 드래그 앤 드롭
```

## 📌 포모도로 기법이란?

이탈리아어로 **"토마토"** 라는 뜻. 1980년대 Francesco Cirillo가 토마토 모양 타이머로 공부하며 고안한 시간 관리 기법입니다.

- 🍅 **25분 집중** → ☕ **5분 휴식** 반복
- 4세션 완료 후 **15~30분 긴 휴식**
